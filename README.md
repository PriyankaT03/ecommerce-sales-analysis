# 🌟 About This Project

This repository presents a complete Amazon sales analytics journey built around data cleaning, exploratory analysis, SQL-based querying, and an interactive Power BI dashboard. It is designed to showcase how raw sales records can be transformed into actionable business intelligence.

The project focuses on understanding revenue trends, customer behavior, product performance, geographic sales distribution, and shipping efficiency through a visual and analytical approach.

---

## 🧠 What Makes This Project Special

- End-to-end analytics workflow from raw CSV data to dashboard storytelling
- Business-focused questions answered with measurable insight
- Clean and professional portfolio-ready presentation
- Strong blend of Python, SQL, and Power BI for modern data analysis

---

## 📌 Core Objectives

- Analyze total sales and revenue performance
- Identify high-performing products and categories
- Understand customer purchasing patterns
- Examine regional and market-level sales trends
- Review payment preferences and shipping behavior
- Create an executive-style dashboard for quick decision-making

---
